WebStorm别名识别配置

sifu   in 南昌
2023-10-09 Monday
  • vue
  • WebStorm
  • 别名

1、项目根目录创建文件：alias.config.js

/**
 * 本文件对项目无任何作用，仅作为 WebStorm 识别别名用
 * */
const path = require('path')
const resolve = dir => path.join(__dirname, dir);
module.exports = {
  resolve: {
    alias: {
      '@': resolve('.')
    }
  }
};
